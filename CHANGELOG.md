# Changelog

所有显著的更改都会记录在此文件中。

## [0.1.0] - 2025-07-09

### 新功能

- 实现视频关键帧提取功能，基于帧间灰度差异自动识别关键帧  
- 支持多宫格拼接关键帧图片，方便快速预览视频内容  
- 支持根据关键帧生成固定时长的视频和 GIF，支持自动控制文件大小和分辨率  
- 提供命令行工具，支持 `extract` 和 `clip` 两个子命令  
- 支持 GIF 循环播放控制和视频编码参数配置  
- 支持输出路径自动生成，简化使用流程  
- 支持详细日志打印，方便调试和使用反馈
